网上找的示例,效果比较全面:
1,可以设置滑动出入的动画效果;
2,可以设置阴影;
3,可以设置左滑动,右滑动,左右滑动;
4,可以集成ViewPager,Fragment;
如下图所示:

Project下载:
Github源码地址:
https://github.com/jfeinstein10/SlidingMenu
导入上面源码中的example 的时候会报错,主要是因为需要其他的library包,按照如下步骤下载文章末尾的1,2,3源码,导入工程到eclipse就好:
1,SlidingMenu 需要actionbarsherlock 包
导入后右键工程 ->Properties --> Android ---> 勾选 Is Library
2,Sliding library
导入后右键工程 ->Properties --> Android ---> Add 把上面的library添加进来,并且本身 勾选 Is Library
3,示例 example project(由于proguard里面的一个txt文档太大了,超过限制,我把他给删除了)
导入后右键工程->Properties --> Android --->Add 把 第二步的 library 添加进来就ok
如果报错的话 可以看具体错误,如下的文章写的比较全面:
http://blog.csdn.net/tj_shenzhendaxue/article/details/8617329
更多UI流行效果:
Android UI Patterns APK下载(内有各个demo源码链接)各种流行UI效果
http://www.eoeandroid.com/forum. ... &fromuid=560113
